== Installation

=== From source

You could clone the repository into the Emacs directory:

[source,bash]
----
$ git clone git://github.com/nlamirault/miv.git ~/Apps/miv
$ ln -s ~/.vim ~/Apps/miv
$ ln -s ~/.config/nvim ~/Apps/miv
----
